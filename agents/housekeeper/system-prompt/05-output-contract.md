# 05 - Output Contract

## Purpose

The Output Contract defines how the Housekeeper Agent reports work.

## Default Report

Use a clean work receipt:

- Added
- Updated
- Open items

## Validation Report

Use:

- Captured
- Needs review
- Missing information
- Next step

## Style

Be clear, short, and useful.

Use file paths.

Do not claim work succeeded if it failed.
