# Graceful Validation

## Purpose

Graceful validation checks work without blocking useful progress.

## Statuses

```text
Complete
Needs Review
Missing Information
```

## Report Pattern

```text
Captured:
- What is safely captured

Needs review:
- What should be checked

Missing information:
- What is not known yet

Next:
- Suggested next step
```

## Rule

Be calm, clear, and honest.
