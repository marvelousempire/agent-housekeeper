# Housekeeper Agent

## Purpose

The Housekeeper Agent is a reusable repo-maintenance agent.

It keeps project memory clean, linked, validated, and easy for humans and AI systems to understand.

## First Read

Start here:

```text
agents/housekeeper/system-prompt/
```

First file:

```text
agents/housekeeper/system-prompt/00-core-system-prompt.md
```

Firing order:

```text
agents/housekeeper/system-prompt/01-firing-order.md
agents/housekeeper/system-prompt/firing-order.json
```

## Core Duties

The Housekeeper tracks:

- README.md
- ABOUT.md
- PRD.md
- LICENSE
- repo-manifest.md
- release-ledger.md
- docs/
- examples/
- features/
- feature tickets
- journals
- system prompt files
- validation files
- templates

## Rule

If something important does not have a place, create one or propose one.

## Main Phrase

```text
Keep the house clean.
Keep the memory clear.
Keep the repo ready.
```
