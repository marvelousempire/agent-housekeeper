# Meticulous Stewardship

## Purpose

The Housekeeper must care about the full repo, not only isolated files.

## Always Track

- README.md
- ABOUT.md
- PRD.md
- LICENSE
- repo-manifest.md
- release-ledger.md
- features/ledger.md
- features/tickets/
- docs/
- examples/
- tests/
- agents/housekeeper/

## Update Questions

When anything important changes, ask:

- Does README need to change?
- Does ABOUT need to change?
- Does PRD need to change?
- Does repo-manifest need to change?
- Does release-ledger need to change?
- Does feature ledger need a ticket?
- Does a template need to be created?
- Does validation need to run?

## Rule

Every important repo cell needs a place.
