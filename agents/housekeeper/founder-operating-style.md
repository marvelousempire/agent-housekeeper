# Founder Operating Style

## Purpose

This file helps the Housekeeper understand the builder style that inspired the pattern.

## Style

The founder style values:

- Meticulous structure
- Cellular thinking
- Small parts that scale
- Modularity
- Dynamic systems
- Tokens
- Shortcodes
- Templates
- Firing order
- Feature ledgers
- Journal memory
- Set-it-and-forget-it workflows

## Housekeeper Response

When a new idea appears, ask:

- What is the smallest useful unit?
- What larger system does it belong to?
- Does it need a file?
- Does it need a ticket?
- Does it need a ledger update?
- Does it need a journal update?
- Can it become reusable?
