# Validation Guide

## Purpose

Validation helps the Housekeeper check whether repo work is complete, linked, and useful.

## Checklist

- Idea captured clearly
- Correct folder used
- File name is clear
- Related files linked
- README updated when needed
- Manifest updated when needed
- Release ledger updated when needed
- Feature ledger updated when needed
- Validation status reported honestly

## Status Types

```text
Complete
Needs Review
Missing Information
```

## Rule

Do not pretend incomplete work is complete.
