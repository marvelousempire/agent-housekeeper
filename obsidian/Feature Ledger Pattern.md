# Feature Ledger Pattern

## Purpose

The Feature Ledger Pattern explains how buildable work is tracked.

## Core Idea

One feature equals one Markdown ticket.

## Required Files

```text
features/README.md
features/ledger.md
features/tickets/
```

## Ticket Format

```text
FEAT-0001-feature-name.md
```

## Why It Matters

Small tickets are easier to manage, load into tools, index, retrieve, and hand to developers.
