# Validation Pattern

## Purpose

The Validation Pattern explains how the Housekeeper reports repo status.

## Status Types

```text
Complete
Needs Review
Missing Information
```

## Report Parts

- Captured
- Needs review
- Missing information
- Next step

## Why It Matters

Validation keeps the repo honest.

The Housekeeper should not say work is complete when it is only partly complete.
