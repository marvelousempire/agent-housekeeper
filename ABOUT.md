# About Agent Housekeeper

Agent Housekeeper is a reusable AI-agent standard for maintaining project repos.

It is designed for founders, builders, and teams who want their repos to become clean, structured, modular, searchable, RAG-ready, wiki-ready, and easy for future agents to understand.

## Why This Exists

Most repos collect files, ideas, notes, and decisions over time.

Without a housekeeper, important context gets lost.

Agent Housekeeper gives an AI agent a clear job:

```text
Keep the house clean.
Keep the memory clear.
Keep the repo ready.
```

## What It Does

Agent Housekeeper defines:

- Agent persona
- System prompt folder
- Firing order
- Knowledge source rules
- Repo transformation guide
- Feature ledger guide
- Journal templates
- Validation rules
- Graceful validation patterns
- Repo audit templates
- Reusable folder structures

## Design Philosophy

Small pieces should scale.

Every important thing should have a place.

Every repo should have a front door, a memory system, a feature ledger, a release ledger, a manifest, and a housekeeper agent that knows where to begin.

## Reusable Standard

This repo can be used as a standard for other projects.

A project can copy the folder structure or point an agent to this repo as the source of Housekeeper behavior.
